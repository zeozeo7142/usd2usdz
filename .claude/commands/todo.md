cTODO.md를 읽고 그 작성 요령에 따라 TODO.md를 작성하거나 업데이트한다.

## 업데이트 원칙 (중요)

기존 `TODO.md`가 있으면 반드시 먼저 읽은 뒤 아래 기준으로 처리한다.

- **변경 없는 내용**: 그대로 유지한다.
- **변경된 내용**: 해당 부분을 새 내용으로 재작성한다. (경로 변경, 명령어 수정, 완료 상태 변경 등)
- **새로 추가된 내용**: 적절한 위치에 새 항목/Step으로 추가한다.

## 작업 순서

1. `cTODO.md`를 읽어 TODO.md 작성 원칙과 구조를 파악한다.
2. 현재 `TODO.md`를 읽어 기존에 기록된 내용과 완료 상태를 파악한다.
3. 이번 대화의 작업을 기존 내용과 비교한다.
   - 내용이 같으면 유지
   - 내용이 바뀌었으면 해당 부분 재작성 (경로, 명령어, 완료 여부 등)
   - 새 작업이면 성공한 것만 추려 적절한 위치에 추가
4. TODO.md를 저장한다.
